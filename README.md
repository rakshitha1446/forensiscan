# ForensiScan 🔍

AI forensic detection web app for identifying AI-generated media.

## Features

• Detects AI generated images
• Multi-vector forensic analysis
• Entropy and frequency artifact detection
• Statistical fingerprint analysis
• Weighted sigmoid classifier

## Tech Stack

Frontend
HTML
CSS
JavaScript

Backend
Node.js
Express
Sharp
Multer

## How it works

1. Upload image/audio/video/text
2. Backend runs forensic detection vectors
3. System calculates AI probability
4. Returns verdict and confidence score

## Project Structure

FRONTEND → UI and upload interface  
BACKEND → forensic analysis engine
